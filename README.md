# Neuvillette-Helper
Neuvillette automatic circle program. 

那维莱特自动滋水程序



## 使用方法

### 编译可执行文件

克隆代码后在本地编译即可

```bash
git clone https://github.com/xiabee/Neuvillette-Helper.git
cd ./Neuvillette-Helper
go mod download

GOOS=windows GOARCH=amd64 go build -o neuvillette.exe .
```



不会编译的话直接去 release 页面下载最新版本可执行文件即可
